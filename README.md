## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites
- Install Node.js
- Install npm
- Install Python
- Install Django
- Install Git
- Install VS Code


### Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/sajan69/CodeFrontier.git
   ```
2. Navigate to the project directory:
   ```sh
   cd CodeFrontier
   ```
