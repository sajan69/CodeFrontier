# Day 8: CSS Styling

## Overview
This lesson covers advanced CSS styling techniques, including backgrounds, borders, gradients, shadows, and CSS variables. These tools enable creating visually appealing and maintainable web designs.

## Key Topics Covered
1. **Backgrounds**
   - Background properties
   - Multiple backgrounds
   - Background patterns
   - Background positioning
   - Background effects

2. **Borders**
   - Border styles
   - Border radius
   - Border images
   - Outline properties
   - Border animations

3. **Gradients**
   - Linear gradients
   - Radial gradients
   - Conic gradients
   - Color stops
   - Gradient patterns

4. **Shadows**
   - Box shadows
   - Text shadows
   - Multiple shadows
   - Inner shadows
   - Shadow effects

5. **CSS Variables**
   - Custom properties
   - Variable scope
   - Fallback values
   - Dynamic theming
   - Media query integration

## Practice Exercise
Create a modern card component showcasing:
- Complex background patterns
- Creative border effects
- Custom gradient backgrounds
- Multiple shadow layers
- Theme switching using CSS variables
